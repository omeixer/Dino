# 🎮 ChiefData Offline Runner - Desktop Edition

A fully-featured desktop runner game with sound effects, particle system, and offline support.

## ✨ Features

- 🎵 **Built-in Sound Effects** (No external audio files needed)
- 💥 **Particle Effects** (Explosions, jump dust)
- 🌓 **Dark/Light Theme Toggle**
- 📊 **Persistent High Scores** (Local Storage)
- ⌨️ **Keyboard & Mouse Controls**
- 🚀 **Progressive Difficulty**
- 📴 **100% Offline Capable**

## 🎮 Controls

| Key | Action |
|-----|--------|
| `SPACE` / `↑` / `W` | Jump |
| `R` | Restart Game |
| Click Canvas | Jump |
| UI Buttons | Game Controls |

## 🚀 Installation

### Method 1: Direct Download
1. Download all files to a folder
2. Open `index.html` in any modern browser
3. Play offline!
